# Apply Flow


# ✨ Features
🔥 One-click job application
📄 Auto-fill application forms
👤 Saves user profile details
📎 Resume upload support
🧠 Smart field detection
⚡ Faster job applications
🌐 Works on multiple job portals
💾 Local storage support
🔐 Secure user data handling

# 🧩 How It Works
User opens a job portal
Extension detects application form
Saved user information is fetched
Fields are automatically filled
User reviews and submits application
